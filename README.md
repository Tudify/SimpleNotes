SimpleNotes is designed to perfectly complement SimpleWeb in every way, while being truly amazing.

## controls

| Action   | shortcut |
|----------|----------|
| Ctrl + S | Save     |
| Ctrl + E | Export   |
| Ctrl + N | New      |

## extra info

- Notes are stored in JSON in your home directory
- SimpleNotes uses PyQt5
